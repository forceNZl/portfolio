# Portfolio de Recrutement 2026 - Mukasa Theophile Kevin

![Aperçu du Portfolio](public/kev.png)

Ce projet est un portfolio personnel à haute conversion conçu spécifiquement pour la recherche de stage en **Ingénierie Logicielle (2026)**. Il met en avant un profil hybride d'**Ingénierie** et d'**Analyse Business**.

## 🚀 Vision du Projet

L'objectif est de transformer un simple CV en une landing page de services stratégiques. Le design est épuré, lumineux et moderne, inspiré des interfaces SaaS haut de gamme.

### Points Forts :
- **Positionnement Stratégique** : Focus sur le ROI et la valeur ajoutée commerciale.
- **Cycle d'Innovation** : Animation fluide du titre pour capter l'attention.
- **Identité Multiculturelle** : Mise en avant du parcours entre Goma (RDC) et Gisenyi (Rwanda).
- **Responsive Design** : Optimisé pour tous les écrans (Mobile, Tablette, Desktop).

## 🛠️ Stack Technique

- **Frontend** : [Vite.js](https://vitejs.dev/) + [Tailwind CSS](https://tailwindcss.com/)
- **Icônes** : [Phosphor Icons](https://phosphoricons.com/)
- **Typographie** : Outfit (Google Fonts)
- **Déploiement** : Prêt pour Vercel / Netlify

## 📁 Structure du Projet

```bash
├── public/          # Assets statiques (Photos, CV PDF)
├── src/             # Logique JavaScript et CSS principal
├── index.html       # Structure sémantique du portfolio
└── tailwind.config.js # Configuration personnalisée (animations, thèmes)
```

## 📋 Installation Locale

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/forceNZl/portfolio.git
   ```
2. Installez les dépendances :
   ```bash
   npm install
   ```
3. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```
